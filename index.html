Topics for Graduation Thesis

Please remember that you are to write a good report on these software. In almost all of them, you are going to show other examples and detailed information about the technologies you've learned and use. It is not just a software project, there is also research.

I want everyone to have a version control system in place, preferably Git.

  • Exam Generator: Task is first to create a very basic DSL. Later, the application will parse the text file and generate a LaTeX document using the exam module.
  • Static Site Generator: Task is to create a static web page based on a given directory and subdirectories. Posts will be in Markdown. Theme capability would be good.
  • Password Manager: Task is to create a password manager. It will store passwords in encrypted form, the key is going to be derived from the master password. User is expected to learn about password storage, encryption, key derivation functions, etc. A GUI application would be better.

    • NOTES

    You are free to propose projects, but keep in mind it is possible for me to reject them. I will not accept too easy projects which can be done in a week. It should not be entirely composed of a single library.

    Projects should be a product. It should just work when I run them, and you should be able to answer any question about the topic and the code you've written. So, even if you are building a data app, I will not accept if you are playing with the code from the Jupyter notebook, it should be adjustable in the UI.

    Whatever the project is, I also want a thorough thesis.
